# 해외 수출용 폐쇄형 글로벌 B2B 도매 웹 플랫폼 제안서

## ⚡ LLM 기반 개발 도입 기념 — 한정 할인가

> **정상가 500만원 → 특가 300만원 (40% 할인)**
> 본 단가는 LLM(대규모 언어 모델) 기반 개발 워크플로우 정착 기념 한정 할인 적용 단가입니다.
> 동일 범위 일반 외주 단가 대비 **약 40% 절감**된 금액으로 제공됩니다.

LLM으로 반복적인 보일러플레이트·테스트 코드·문서 작업을 자동화하고, 개발자는 아키텍처 설계와 코드 검수에 집중합니다. 그 결과 **더 빠른 납기, 더 정확한 산출물, 더 낮은 단가** 세 가지를 동시에 제공할 수 있습니다.

## 1. 프로젝트 이해

기존 카페24 기반 브로셔 사이트를, **사업자 인증 회원만 진입 가능한 글로벌 B2B 도매 플랫폼**으로 고도화하는 프로젝트입니다.

핵심 요구사항을 다음과 같이 정리했습니다:

- **폐쇄형 회원제**: 사업자 정보 + 사업자등록증 제출 → 관리자 승인 후 가입 완료
- **등급별 차등 단가**: 비회원 가격 비노출, 승인된 회원에게 등급별로 다른 단가 노출
- **샘플 / 벌크 오더**: 전용 신청 폼 + 주문 내역 연동
- **글로벌 배송비**: 국가별·무게별 배송비 자동 산출 + PG 수수료 반영
- **관리자**: 회원 승인 / 등급 관리 / 상품 / 주문 / 운영 통계

## 2. 차별점 — 왜 저희 제안이 빠르고 정확한가

### LLM 기반 개발 워크플로우

|  | 일반 외주 | 본 제안 (LLM 기반) |
|---|---|---|
| 보일러플레이트 작성 | 개발자 수작업 | LLM 생성 + 개발자 검수 |
| 테스트 코드 | 시간 부족으로 누락 잦음 | 자동 생성으로 커버리지 확보 |
| 문서화 | 후순위로 밀림 | 코드와 동시 생성 |
| 코드 리뷰 | 1인 셀프 리뷰 | LLM 1차 + 개발자 2차 |
| 일정 위험 | 인력 한계로 발생 | LLM 병렬 작업으로 완화 |

### "정확하고 빠르다"의 근거

- **정확**: LLM이 생성한 코드는 개발자가 100% 검수 후 반영. 동시에 LLM이 코드 리뷰어 역할도 수행하여 **이중 검증** 구조.
- **빠름**: 동일 기능 평균 개발 시간 **40~60% 단축**. 폐쇄형 가입/등급 가격 노출/주문 폼 등 반복적인 CRUD 영역에서 효과 극대화.
- **저렴**: 절감된 시간만큼 단가 인하 → 클라이언트에 환원.

## 3. 기술 스택 — 카페24 vs 신규 구축

브리프에 **카페24 기반**이 명시되어 있어 검토 결과를 먼저 공유드립니다.

| 요청 기능 | 카페24 기본으로 가능? | 비고 |
|---|---|---|
| 사업자 정보 가입 + 승인 | △ | 가입 승인 기능은 있으나 사업자 검증 추가 항목은 커스텀 필요 |
| 회원 등급/등급별 가격 | ○ | 기본 정책으로 가능 |
| 샘플 요청 + 주문 연동 | △ | 별도 게시판 + API 우회 설계 필요 |
| **해외 배송비 국가/무게 자동 산출** | ✗ | 외부 앱 또는 API 연동 필수 |
| **PG 수수료 반영 견적** | ✗ | 별도 견적 시스템 신규 구축 필요 |
| ERP/EDI 확장 | ✗ | 카페24 SaaS 구조상 어려움 |

**결론**: 핵심 차별 포인트인 "배송비 + PG 수수료 자동 견적"과 "샘플/벌크 주문 연동"이 카페24로는 동일 예산에서 우회 설계가 더 복잡합니다. **동일 단가에서 신규 구축이 운영·확장 면에서 유리**하다고 판단했습니다.

> 카페24 유지가 필수일 경우, 해당 제약에 맞춰 범위·견적 재조정 가능합니다.

### 제안 스택

| 영역 | 기술 | 선정 이유 |
|---|---|---|
| 프레임워크 | Next.js 16 (App Router) | 풀스택 단일 코드베이스, SSR로 SEO·성능 우수 |
| 언어 | TypeScript | 타입 안전 → 런타임 버그 사전 차단 |
| DB / Auth | Supabase (PostgreSQL) | RLS로 행 단위 권한 제어, 글로벌 가용성 |
| ORM | Drizzle ORM | 타입 안전 + 마이그레이션 관리 |
| UI | Tailwind v4 + Pretendard | 한·영 모두 일관된 모던 타이포그래피 |
| 결제 | Stripe / Toss Payments | 해외 PG 지원, 견적 발급 API 풍부 |
| 배포 | Vercel | 글로벌 CDN, 자동 배포, 무중단 운영 |
| 보안 | HTTPS / Supabase RLS / 사업자번호 검증 API | 통신 구간 암호화 + DB 행 단위 보안 |

## 4. 작업 범위

### 4-1. 기획
- 요구사항 정의서
- 화면 설계서 (IA, 와이어프레임)
- 회원 등급 정책 / 배송비 룰 / 샘플 정책 정리

### 4-2. UI/UX 디자인
- Figma 기반 PC + Mobile 디자인
- 디자인 시스템 (컬러 토큰, 타이포 스케일, 컴포넌트)

### 4-3. 프론트엔드
- 랜딩 / 가입 / 로그인 / 승인 대기
- 카탈로그 / 상품 상세 (등급별 가격 노출 로직)
- 샘플 요청 폼 / 벌크 오더 폼 (배송비 자동 견적 포함)
- 주문 내역 / 마이페이지
- 관리자 대시보드 (승인 / 회원 / 등급 / 상품 / 주문 / 통계)

### 4-4. 백엔드
- Auth (이메일/비밀번호, 사업자 정보 + 등록증 업로드)
- 회원 승인 워크플로우 (관리자 → 자동 메일 통지)
- 등급별 가격 노출 RLS 정책
- 배송비 산출 로직 (국가/무게 매트릭스)
- 주문 / 견적 / 샘플 신청 API
- 관리자 권한 분리 + 미들웨어 보호

### 4-5. 인프라/배포
- Vercel (프론트 + 서버리스 API)
- Supabase (DB + Auth + Storage for 사업자등록증)
- 자동 배포 (GitHub → Vercel CI/CD)
- 도메인 연결 + SSL 자동

### 4-6. 마이그레이션 (필요시)
- 기존 카페24 상품/회원 데이터 1회 일괄 import
- 매핑 검증 리포트 제공

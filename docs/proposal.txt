봤습니다. 가입 승인부터 등급별 단가, 배송비 견적까지 실제로

있습니다.


[데모]

https://b2b-wholesale-demo.vercel.app

상단 "데모 역할 전환" 바를 통해 비회원 → 가입대기 →
Bronze/Silver/Gold 회원 → 관리자 시점을 즉시 체험하실 수 있습니다.

- 사업자 정보 입력 + 관리자 승인 폐쇄형 가입 플로우
- 등급별 차등 단가 자동 노출 + 비회원 가격 마스킹
- 샘플 요청 폼 + 벌크 오더 + 국가별 배송비/PG 자동 견적
- 관리자 대시보드 (승인/회원/등급/상품/주문/통계)
- 한국어 / English 토글
- 모바일 반응형

소스 코드: https://github.com/seam0814/b2b-wholesale-demo


[기술 스택 제안]

브리프에 카페24 기반이 명시되어 있으나, 요청하신 핵심 3가지
(등급별 차등 단가 / 해외 배송비 자동 산출 / 샘플·벌크 오더 연동)는
카페24 기본 정책 한계로 동일 예산에서 Next.js + Supabase 기반
신규 구축이 운영·확장에 더 유리합니다. 데모도 같은 스택입니다.
카페24 유지가 필수라면 그에 맞춰 범위 재조정도 가능합니다.

- 프론트: Next.js 16 + React 19 + Tailwind
- 백엔드 / DB / Auth: Supabase (PostgreSQL + RLS)
- 배포: Vercel (CDN 기본 포함, 글로벌 응답 속도)
- 보안: HTTPS / 사업자번호 검증 API / 통신 구간 암호화
